# YouTube Channel Hunter Frontend

This frontend is a clean Vite + React + Tailwind starter for YouTube Channel Hunter v1.2.

## Setup

- Install dependencies: npm install
- Start the dev server: npm run dev

## Structure

- src/components for reusable UI pieces
- src/pages for page-level views
- src/assets for local static assets

This app is intentionally independent from the existing FastAPI backend and MCP folder so it can be connected later.
